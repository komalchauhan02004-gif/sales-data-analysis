# Sales Data Analysis Project

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Analysis Performed
- Loaded and cleaned sales data
- Calculated total sales and total profit
- Performed category-wise sales analysis
- Created bar chart visualization

## Key Insights
- Technology category has highest sales
- Office Supplies has steady performance
- Furniture shows mixed profitability

## Files Included
- `Sales_Data_Analysis.ipynb` → Jupyter Notebook with code and analysis
- `cleaned_sales_data.csv` → Original dataset
